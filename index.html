Current focus

Research

My work asks how intelligent software can remain inspectable, verifiable and accountable.

01

Trustworthy AI for code

Code provenance, watermarking, automated repair and rigorous evaluation of AI-assisted development.

02

Program analysis & security

Static, symbolic and hybrid analysis for C, WebAssembly, Android software and smart contracts.

03

Responsible software intelligence

Compliance benchmarks, software agents and reproducible methods for evidence-driven software engineering.
